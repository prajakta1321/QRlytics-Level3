# SmartQR – Level 3 (QR Management System)

SmartQR Level 3 is a web-based QR Code Management System developed using Python, Streamlit, and SQLite. It allows users to generate customizable QR codes, store generation history in a database, preview generated QR codes, and download them directly from the web interface.

##  Features

### QR Code Generation
- Generate QR codes from text or URLs
- Real-time QR code preview
- Download generated QR code as PNG

### QR Customization
- Choose custom QR color
